# PhishGuard AI

PhishGuard AI is a machine learning project that detects phishing attempts from URLs and email/message text.

## Features
- URL phishing detection
- Email/SMS phishing detection
- Confidence score
- Simple explainable output
- Streamlit user interface

## Tech Stack
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Random Forest
- Logistic Regression
- Streamlit

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. run app.py
python -m streamlit run app.py
